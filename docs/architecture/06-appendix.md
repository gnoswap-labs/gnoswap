# Appendix

## A. Contract Addresses (Roles)

| Role | Description |
|------|-------------|
| admin | System administrator |
| governance | Governance contract |
| pool | Pool domain |
| position | Position domain |
| router | Router domain |
| staker | Staker domain |
| emission | Emission contract |
| gns | GNS token |
| protocol_fee | Protocol fee |

## B. Fee Structure

| Fee | Value | Description |
|-----|-------|-------------|
| Pool Creation | 100 GNS | Pool creation cost |
| Swap Fee Tiers | 0.01~1% | Per-pool swap fees |
| Protocol Fee | 0-10% | Protocol share of swap fees |
| Router Fee | 0.15% | Router fee |
